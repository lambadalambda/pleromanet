Postcards — four variants

Pre-made frames you drop a photo and 1-line caption into, then post. All use the duotone filter so the image always agrees with the theme, and lean on serif + handwriting fonts for that "sent from somewhere" feel. The thumbnails at the bottom of each row show how it would render inline in the feed.
